Цель работы:
Разработать веб-версию игры 2048 с использованием только чистого HTML, CSS и JavaScript без применения сторонних фреймворков. Реализовать основную игровую механику, систему подсчёта очков, сохранение состояния и таблицу рекордов.

Реализованный функционал:

Динамическая генерация игрового поля 4×4

Управление:стрелками на клавиатуре (ПК),виртуальными кнопками, свайпами по игровому полю(мобилки)

Логика движения и слияния плиток во всех направлениях

Появление 1–2 новых плиток со значениями 2 или 4 после каждого хода

Подсчёт текущего счёта и отображение лучшего результата

Проверка окончания игры (отсутствие возможных ходов)

Кнопка «Начать заново»

Кнопка Undo (отмена одного хода)

Сохранение состояния игры в localStorage

Таблица лидеров (Top-10) с сохранением:имени игрока,количества очков,даты сохранения

Модальное окно при завершении игры с возможностью ввода имени

Адаптивный дизайн для ПК, планшетов и смартфонов

Цветовая индикация плиток по значениям

Анимация появления плиток
